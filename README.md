# Modrinth-AI-Blocklist
uBlockOrigin blocklist of ai slop on modrinth.com

# Installing

If you have uBlock Origin installed, click [this link](https://subscribe.adblockplus.org?location=https%3A%2F%2Fraw.githubusercontent.com%2FMayaqqDev%2FModrinth-AI-Blocklist%2Fmain%2Flist.txt&title=Mods%20on%20Modrinth%20using%20AI%20generated%20content) to import the filter list in just a click! Quick and simple.

# Criteria
Creator:
- 1/3 or more of their mods use ai generated content, criteria as below
Icon:
- Icon is AI generated
- Provide reasons for such assumption if submitting an issue/pr
Description:
- Clearly an AI generated description
- Signs of AI such as formatting, wordslop, etc
Code:
- AI comments, CLAUDE folder, etc
- Provide snippets and links to why you think it is AI
